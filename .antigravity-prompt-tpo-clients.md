# Task: Fix TPO "Client Directory" dead end (`/tpo/clients`)

## The bug
`app/tpo/clients/page.tsx` renders `<ClientSearch role="tpo" title="Clients Directory"
description="View companies hiring from your college" />`. The shared component
`components/dashboard/client-search.tsx` only fetches data when `user.role === 'ADMIN'`
(line 30: `if (user && user.role === 'ADMIN') { fetchClients() }` — the `else` branch just
sets `loading=false` with a comment "Assuming TPO might not have /admin/users access").
The search bar is also hidden for non-admins (line 71: `{user.role === 'ADMIN' && (...)}`).
Net effect: every TPO permanently sees "No clients found", with no search, regardless of
real data. This is NOT the same use case as the admin's global client list — the page's
own copy promises "companies hiring from your college", which is a genuinely different,
narrower query: which clients have a Contract with a student from this TPO's college.

## Don't reuse the admin endpoint — build a TPO-scoped one
`GET /admin/users?role=CLIENT` lists ALL clients platform-wide with no college relevance.
That's wrong for this page even if TPOs were given access to it. Build a real, scoped
endpoint instead.

## Backend
Schema path is confirmed: `Contract.studentId -> Student.collegeId` and
`Contract.projectId -> Project.clientId -> Client`. Add a method to
`backend/src/modules/tpo/tpo.service.ts` following the exact pattern already used by
every other method in that file (e.g. the existing student-list method around line 13-22
and `assertOwnCollegeStudent` around line 82-92): look up the TPO by `userId` to get
`tpo.collegeId`, then query:

```ts
async getHiringClients(tpoUserId: string) {
  const tpo = await this.prisma.tPO.findUnique({ where: { userId: tpoUserId } });
  if (!tpo) throw new ForbiddenException();

  const contracts = await this.prisma.contract.findMany({
    where: { student: { collegeId: tpo.collegeId } },
    include: {
      project: {
        include: { client: { include: { user: { select: { name: true, avatarUrl: true } } } } },
      },
    },
    distinct: ['projectId'], // dedupe if a client hired multiple students from this college
  });

  // De-dupe by client.id (a client may appear via multiple contracts) and shape the response
  const seen = new Map();
  for (const c of contracts) {
    const client = c.project.client;
    if (!seen.has(client.id)) {
      seen.set(client.id, {
        id: client.id,
        name: client.user.name,
        avatarUrl: client.user.avatarUrl,
        companyName: client.companyName,
        industry: client.industry,
        projectsHired: 0,
      });
    }
    seen.get(client.id).projectsHired += 1;
  }
  return Array.from(seen.values());
}
```//

Add a controller route in `backend/src/modules/tpo/tpo.controller.ts`, following the
